Installation guide
  1. 1 Tap Download FlipTok APK above. Your browser saves fliptok.apk to Downloads.
  2. 2 Go to Settings → Apps → [your browser] → Install unknown apps and tap Allow.
  3. 3 Open Downloads, tap fliptok.apk, then tap Install.
  4. 4 Get a free YouTube Data API v3 key and a Gemini API key. Add them in Settings → API Keys.
  5. 5 Set your genre and length preferences, then start watching.
