Installation guide
  1. 1Tap Download FlipTok APK above. Your browser saves fliptok.apk to Downloads.
  2. 2Go to Settings → Apps → [your browser] → Install unknown apps and tap Allow.
  3. 3Open Downloads, tap fliptok.apk, then tap Install.
  4. 4Grab a free YouTube Data API v3 key and a Gemini API key, then add them in Settings → API Keys.
  5. 5Set your genre and length preferences, then start watching.
